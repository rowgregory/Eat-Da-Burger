# Eat-Da-Burger

 * Eat-Da-Burger is a burger logger built with MySQL, Node, Express, Handlebars and a homemade ORM (yum!).

## Getting Started

 * Install node.js/npm
 * Clone repo
 * Run NPM install
 * Set up mysql database and run schema/seed files
 * node server.js
 * localhost:3000 on your browser
